# Token Airdrop Contract

An example ERC20 airdrop script written in Solidity for token distribution.

## Features
- Batch token send
- Compatible with any ERC20 token
- Lightweight code

## Technologies
Solidity  
ERC20 standard  
Ethereum Base network
